# nalogo-go

Go-клиент к API сервиса ФНС «Мой налог» (lknpd.nalog.ru) для самозанятых.

Порт асинхронной Python-библиотеки [rusik636/nalogo](https://github.com/rusik636/nalogo) на Go в виде переиспользуемой библиотеки.

## Status

Скелет. API не реализован.

## Scope (план переноса)

- Авторизация: ИНН + пароль, SMS, refresh токенов
- Регистрация дохода (одна и несколько позиций)
- Аннулирование чеков
- Получение чека (JSON, print URL)
- Профиль, способы оплаты
- История налогов / OKTMO

## Caveats

- API «Мой налог» неофициальное (реверс мобильного приложения), может меняться без предупреждения.
- Все строки/ошибки/маскирование чувствительных данных — по поведению апстрима.
